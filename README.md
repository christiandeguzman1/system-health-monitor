#Real-Time System Health Monitor

A real-time system monitoring dashboard built with Python, Streamlit, and psutil.

## Features
- Live CPU, Memory, and Disk usage tracking
- Auto-refreshing dashboard
- Historical usage charts
- Top running processes by CPU and Memory
- Alert system for high resource usage
- Export monitoring data to CSV

## Tech Stack
- Python
- Streamlit
- psutil
- Pandas

## Installation

```bash
pip install streamlit psutil pandas
streamlit run app.py
